# Carmen-Lu.github.io
A background monitoring and management website
项⽬基于vue2、echarts和koa开发。
实现项目响应式布局；
vue-router实现单⻚⾯组件开发并进行组合；
在了解koa后台开发原理基础上，运用Axios对后台发起数据请求；
并使用Echarts动态进⾏数据统计结果的可视化展示；
使用WebSocket实现客户端联动全屏效果，使项目更贴合实际情况。
